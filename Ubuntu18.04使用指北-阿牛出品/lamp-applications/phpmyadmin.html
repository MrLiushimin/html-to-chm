phpMyAdmin的

phpMyAdmin是专门为管理MySQL服务器而编写的LAMP应用程序 。phpMyAdmin 用PHP编写，可通过Web浏览器访问，为数据库管理任务提供图形界面。

  • 安装
  • 组态
  • 参考

安装

在安装phpMyAdmin之前，您需要在与安装phpMyAdmin 相同的主机上或在可通过网络访问的主机上访问MySQL数据库。有关更多信息，请参阅MySQL。从终端提示输入： 

sudo apt安装phpmyadmin

在提示符下选择要为phpMyAdmin配置的Web服务器。本节的其余部分将使用Apache2作为Web服务器。

在浏览器中转到http：// servername / phpmyadmin，将servername替换 为服务器的实际主机名。在登录时，页面中输入根的用户名，或另一个MySQL的用户，如果您有任何设置，然后输入MySQL的用户的密码。

登录后，您可以根据需要重置root密码，创建用户，创建/销毁数据库和表等。

组态

phpMyAdmin 的配置文件位于/ etc / phpmyadmin中。主配置文件是/etc/phpmyadmin/config.inc.php。此文件包含全局应用于phpMyAdmin的配置选项。

要使用phpMyAdmin管理托管在另一台服务器上的MySQL数据库，请在/etc/phpmyadmin/config.inc.php中调整以下内容： 

$ cfg ['Servers'] [$ i] ['host'] ='db_server';

将db_server替换为实际的远程数据库服务器名称或IP地址。此外，请确保phpMyAdmin主机具有访问远程数据库的权限。

配置完成后，退出phpMyAdmin并重新登录，您应该访问新服务器。

该config.header.inc.php和config.footer.inc.php文件用于添加HTML页眉和页脚phpMyAdmin的。

另一个重要的配置文件是/etc/phpmyadmin/apache.conf，该文件符号链接到 /etc/apache2/conf-available/phpmyadmin.conf，一旦启用，用于配置Apache2 以服务phpMyAdmin站点。该文件包含用于加载PHP，目录权限等的指令 。从终端类型： 

sudo ln -s /etc/phpmyadmin/apache.conf /etc/apache2/conf-available/phpmyadmin.conf 
sudo a2enconf phpmyadmin.conf 
sudo systemctl reload apache2.service

有关配置Apache2的更多信息， 请参阅HTTPD - Apache2 Web服务器。
