WordPress的

Wordpress是一个博客工具，发布平台和CMS，用PHP实现，并在GNU GPLv2下获得许可。

  • 安装
  • 组态
  • 参考

安装

要安装WordPress，请在命令提示符中运行以下命令： 

    sudo apt安装wordpress

您还应该安装apache2 Web服务器和mysql服务器。有关安装apache2 Web服务器的信息，请参阅HTTPD - Apache2 Web Server部分中的安装 子部分。有关安装mysql服务器的信息，请参阅MySQL部分的安装子 部分。

组态

要配置第一个WordPress 应用程序，请配置apache站点。打开/etc/apache2/sites-available/wordpress.conf并编写以下行： 

        Alias / blog / usr / share / wordpress
        <目录/ usr / share / wordpress>
            选项FollowSymLinks
            AllowOverride限制选项FileInfo
            DirectoryIndex index.php
            订单允许，拒绝
            允许所有人
        </目录>
        <目录/ usr / share / wordpress / wp-content>
            选项FollowSymLinks
            订单允许，拒绝
            允许所有人
        </目录>

启用这个新的WordPress网站

    sudo a2ensite wordpress

配置apache2 Web服务器并为WordPress 应用程序做好准备后，应该重新启动它。您可以运行以下命令来重新启动 apache2 Web服务器： 

sudo systemctl重启apache2.service

为了便于多个WordPress安装，此配置文件的名称基于HTTP请求的主机头。这意味着您可以通过简单地将此配置的主机名部分与Apache虚拟主机匹配来为每个VirtualHost进行配置。例如/etc/wordpress/config-10.211.55.50.php,/etc/wordpress/config-hostalias1.php等。这些说明假设您可以通过localhost主机名访问Apache（可能通过使用ssh隧道），如果没有，请替换/etc/wordpress/config-localhost.php，带/etc/wordpress/config-NAME_OF_YOUR_VIRTUAL_HOST.php。

编写配置文件后，您可以为每个WordPress 数据库实例选择mysql的用户名和密码约定 。本文档仅显示一个localhost示例。

现在配置WordPress以使用mysql数据库。打开/etc/wordpress/config-localhost.php文件并编写以下行： 

<？PHP
define（'DB_NAME'，'wordpress'）;
define（'DB_USER'，'wordpress'）;
define（'DB_PASSWORD'，'yourpasswordhere'）;
define（'DB_HOST'，'localhost'）;
define（'WP_CONTENT_DIR'，'/ usr / share / wordpress / wp-content'）;
？>

现在创建这个mysql数据库。使用mysql命令wordpress.sql打开一个临时文件， 并写下以下行： 

创建数据库wordpress;
GRANT SELECT，INSERT，UPDATE，DELETE，CREATE，DROP，ALTER
在wordpress上。*
到wordpress @ localhost
通过'yourpasswordhere'识别;
FLUSH特权;

执行这些命令。

cat wordpress.sql | sudo mysql --defaults-extra-file = / etc / mysql / debian.cnf

现在可以通过访问http：//localhost/blog/wp-admin/install.php来配置 新的WordPress。（或者http：//NAME_OF_YOUR_VIRTUAL_HOST/blog/wp-admin/install.php， 如果您的服务器没有GUI，并且您 通过在另一台计算机上运行的Web浏览器完成WordPress配置。）填写网站标题，用户名，密码和单击电子邮件，然后单击安装WordPress。

记下生成的密码（如果适用）并单击登录密码。您的 WordPress现在可以使用了。
