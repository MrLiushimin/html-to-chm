概观

LAMP安装（Linux + Apache + MySQL + PHP / Perl / Python）是Ubuntu服务器的流行设置。使用LAMP应用程序堆栈编写了大量的开源应用程序。一些流行的LAMP应用程序是Wiki，内容管理系统和管理软件，如phpMyAdmin。

LAMP的一个优点是对不同的数据库，Web服务器和脚本语言具有很大的灵活性。MySQL的流行替代品包括PostgreSQL和SQLite。Python，Perl和Ruby也经常被用来代替PHP。虽然Nginx，Cherokee和Lighttpd可以取代Apache。

最快的入门方法是使用tasksel安装LAMP 。Tasksel是一个Debian / Ubuntu工具，可以将多个相关软件包作为协调的“任务”安装到您的系统上。要安装LAMP服务器：

  • 在终端提示符处输入以下命令： 

    sudo tasksel安装lamp-server

安装后，您将能够以这种方式安装大多数LAMP应用程序：

  • 下载包含应用程序源文件的存档。

  • 解压缩归档文件，通常在Web服务器可访问的目录中。

  • 根据提取源的位置，配置Web服务器以提供文件。

  • 配置应用程序以连接到数据库。

  • 运行脚本或浏览到应用程序的页面，以安装应用程序所需的数据库。

  • 完成上述步骤或类似步骤后，即可开始使用该应用程序。

使用这种方法的一个缺点是应用程序文件没有以标准方式放在文件系统中，这可能会导致混淆应用程序的安装位置。另一个更大的缺点是更新应用程序。发布新版本时，需要使用与安装应用程序相同的过程来应用更新。

幸运的是，许多LAMP应用程序已经为Ubuntu打包，并且可以像非LAMP应用程序一样进行安装。但是，根据应用程序，可能需要一些额外的配置和设置步骤。

本节介绍如何安装一些LAMP应用程序。
