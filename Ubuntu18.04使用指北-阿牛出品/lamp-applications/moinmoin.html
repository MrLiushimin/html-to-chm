Moin Moin

MoinMoin是一个用Python实现的wiki引擎，基于PikiPiki Wiki引擎，并在GNU GPL下获得许可。

  • 安装
  • 组态
  • 验证
  • 参考

安装

要安装MoinMoin，请在命令提示符中运行以下命令： 

sudo apt安装python-moinmoin

您还应该安装apache2 Web服务器。有关安装apache2 Web服务器的信息，请参阅HTTPD中的安装 子部分- Apache2 Web服务器部分。

组态

要配置您的第一个wiki应用程序，请运行以下命令集。我们假设您正在创建一个名为mywiki的wiki ： 

cd / usr / share / moin 
sudo mkdir mywiki 
sudo cp -R data mywiki 
sudo cp -R underlay mywiki 
sudo cp server / moin.cgi mywiki 
sudo chown -R www-data：www-data mywiki 
sudo chmod -R ug + rwX mywiki 
sudo chmod -R o-rwx mywiki

现在你应该配置MoinMoin来找到你的新wiki mywiki。要配置 MoinMoin，请打开 /etc/moin/mywiki.py文件并更改以下行： 

data_dir ='/ org / mywiki / data'

至 

data_dir ='/ usr / share / moin / mywiki / data'

另外，在data_dir选项下面添加data_underlay_dir： 

data_underlay_dir = '的/ usr /共享/莫因/ mywiki /底衬'

如果/etc/moin/mywiki.py文件不存在，则应将/usr/share/moin/config/wikifarm/mywiki.py文件复制到 /etc/moin/mywiki.py文件并执行上述更改。

如果你命名你的wiki作为my_wiki_name你应该插入线“（‘my_wiki_name’河‘*’）”在 /etc/moin/farmconfig.py行后文件 “（‘mywiki’河‘*’ ）“。

一旦你配置了MoinMoin来找到你的第一个wiki应用程序mywiki，你应该配置apache2并为你的wiki做好准备。

您应该在“<VirtualHost *>”标记内的/etc/apache2/sites-available/000-default.conf文件中添加以下行 ： 

### moin
  ScriptAlias / mywiki“/usr/share/moin/mywiki/moin.cgi”
  alias / moin_static <version>“/ usr / share / moin / htdocs”
  <目录/ usr / share / moin / htdocs>
  订单允许，拒绝
  允许所有人
  </目录>
### end moin

上例中的版本是通过运行来确定的： 

$ moin --version

如果输出显示版本1.9.7，则第二行应为： 

alias / moin_static197“/ usr / share / moin / htdocs”

配置apache2 Web服务器并为wiki应用程序做好准备后，应重新启动它。您可以运行以下命令来重新启动 apache2 Web服务器： 

sudo systemctl重启apache2.service

验证

您可以通过将Web浏览器指向以下URL来验证Wiki应用程序并查看它是否有效： 

HTTP：//本地主机/ mywiki

有关更多详细信息，请参阅MoinMoin网站。
