该贴切命令是一个功能强大的命令行工具，它与Ubuntu的工作高级包装工具（APT）为安装新的软件包执行这样的功能，现有的软件包的升级，包列表索引的更新，甚至是升级整个Ubuntu的系统。

作为一个简单的命令行工具，apt与Ubuntu中为服务器管理员提供的其他包管理工具相比具有许多优势。这些优点中的一些包括易于使用而不是简单的终端连接（SSH），以及在系统管理脚本中使用的能力，而系统管理脚本又可以由cron调度实用程序自动化。

apt实用程序 的常用用法的一些示例：

  • 安装包：使用apt工具安装包非常简单。例如，要安装网络扫描程序nmap，请键入以下内容： 

    sudo apt install nmap

  • 删除包：删除包（或包）也很简单。要删除上一示例中安装的软件包，请键入以下内容： 

    sudo apt删除nmap

    多个包：您可以指定要安装或删除的多个包，以空格分隔。

    此外，向apt remove添加--purge选项也将删除包配置文件。这可能是也可能不是预期的效果，因此请谨慎使用。

  • 更新包索引：APT包索引本质上是来自/etc/apt/sources.list文件和/etc/apt/sources.list.d目录中定义的存储库的可用包的数据库。要使用存储库中所做的最新更改来更新本地程序包索引，请键入以下内容： 

    sudo apt update

  • 升级包：随着时间的推移，您的计算机上当前安装的软件包的更新版本可能会从软件包存储库中获得（例如安全更新）。要升级系统，请首先按照上面的说明更新软件包索引，然后键入： 

    sudo apt升级

    有关升级到新Ubuntu版本的信息，请参阅升级。

apt命令的 操作（例如安装和删除软件包）记录在/var/log/dpkg.log日志文件中。
