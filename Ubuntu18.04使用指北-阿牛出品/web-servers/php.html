PHP - 脚本语言

PHP是一种适用于Web开发的通用脚本语言。PHP脚本可以嵌入到HTML中。本节介绍如何使用Apache2和MySQL在Ubuntu系统中安装和配置PHP。

本节假定您已安装并配置了Apache2 Web Server和MySQL数据库服务器。您可以参考本文档中的Apache2和MySQL部分分别安装和配置Apache2和MySQL。

安装

PHP在Ubuntu Linux中可用。与安装在基本系统中的python和perl不同，必须添加PHP。

  • 要安装PHP和Apache PHP模块，可以在终端提示符下输入以下命令： 

    sudo apt安装php libapache2-mod-php

    您可以在终端提示符下运行PHP脚本。要在终端提示符下运行PHP脚本，您应该安装 php-cli软件包。要安装 php-cli，您可以在终端提示符下输入以下命令： 

    sudo apt安装php-cli

    您也可以在不安装Apache PHP模块的情况下执行PHP脚本。要完成此任务，您应该安装 php-cgi包。您可以在终端提示符下运行以下命令来安装 php-cgi包： 

    sudo apt安装php-cgi

    要在PHP中使用MySQL，您应该安装php-mysql包。要安装 php-mysql，您可以在终端提示符下输入以下命令： 

    sudo apt安装php-mysql

    同样，要使用PostgreSQL和PHP，你应该安装php-pgsql包。要安装 php-pgsql，您可以在终端提示符下输入以下命令： 

    sudo apt安装php-pgsql

组态

如果已安装 libapache2-mod-php或 php-cgi软件包，则可以从Web浏览器运行PHP脚本。如果已安装 php-cli软件包，则可以在终端提示符下运行PHP脚本。

默认情况下， 安装libapache2-mod-php时，Apache 2 Web服务器配置为运行PHP脚本。换句话说，安装模块时，Apache Web服务器中启用了PHP模块。请验证文件 /etc/apache2/mods-enabled/php7.0.conf和 /etc/apache2/mods-enabled/php7.0.load是否 存在。如果它们不存在，则可以使用a2enmod命令启用该模块。

一旦安装了PHP相关软件包并启用了Apache PHP模块，就应该重新启动Apache2 Web服务器以运行PHP脚本。您可以在终端提示符处运行以下命令以重新启动Web服务器： 

sudo systemctl重启apache2.service

测试

要验证安装，可以运行以下PHP phpinfo脚本：

您可以将内容保存在文件 phpinfo.php中，并将其放在Apache2 Web服务器的DocumentRoot目录下。将浏览器指向 http：//hostname/phpinfo.php将显示各种PHP配置参数的值。
