Ruby on Rails

Ruby on Rails是一个开源Web框架，用于开发数据库支持的Web应用程序。它针对程序员的可持续生产力进行了优化，因为它允许程序员通过支持约定优于配置来编写代码。

  • 安装
  • 组态
  • 参考

安装

在安装Rails之前，您应该安装Apache 和MySQL。要安装Apache软件包，请参阅 HTTPD - Apache2 Web服务器。有关安装MySQL的说明， 请参阅 MySQL。

一旦安装了Apache和 MySQL软件包，就可以安装Ruby on Rails软件包了。

要安装Ruby基本软件包和 Ruby on Rails，可以在终端提示符中输入以下命令： 

sudo apt安装rails

组态

修改/etc/apache2/sites-available/000-default.conf 配置文件以设置域。

首先要改变的是DocumentRoot 指令： 

DocumentRoot / path / to / rails / application / public

接下来，将<Directory“/ path /更改为/ rails / application / public”>指令： 

<Directory“/ path / to / rails / application / public”>
        选项索引FollowSymLinks MultiViews ExecCGI
        AllowOverride All
        订单允许，拒绝
        允许所有人
        AddHandler cgi-script .cgi
</目录>

您还应该 为Apache 启用mod_rewrite模块。要启用mod_rewrite 模块，请在终端提示符中输入以下命令： 

sudo a2enmod重写

最后，您需要将/ path /到/ rails / application / public 和/ path /到/ rails / application / tmp目录的所有权更改为用于运行Apache进程的用户 ： 

sudo chown -R www-data：www-data / path / to / rails / application / public 
sudo chown -R www-data：www-data / path / to / rails / application / tmp

而已！现在，您的服务器已准备好用于 Ruby on Rails应用程序。
