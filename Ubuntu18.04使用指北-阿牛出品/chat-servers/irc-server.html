IRC服务器

Ubuntu存储库有许多Internet Relay Chat服务器。本节介绍如何安装和配置原始IRC服务器ircd-irc2。

  • 安装
  • 组态
  • 参考

安装

要安装ircd-irc2，请在命令提示符中运行以下命令： 

sudo apt install ircd-irc2

配置文件存储在 / etc / ircd目录中。这些文档位于/ usr / share / doc / ircd-irc2 目录中。

组态

IRC设置可以在配置文件 /etc/ircd/ircd.conf中完成。您可以通过编辑以下行在此文件中设置IRC主机名： 

M：irc.localhost :: Debian ircd默认配置:: 000A

请确保为IRC主机名添加DNS别名。例如，如果将irc.livecipher.com设置为IRC主机名，请确保irc.livecipher.com可在您的域名服务器中解析。IRC主机名不应与主机名相同。

可以通过编辑以下行来配置IRC管理员详细信息： 

答：组织，IRC部门：Daemon <ircd@example.irc.org>：客户端服务器:: IRCnet：

您应该添加特定行来配置要侦听的IRC端口列表，配置操作员凭据，配置客户端身份验证等。有关详细信息，请参阅示例配置文件 / usr / share / doc / ircd-irc2 / ircd .conf.example.gz。

当用户连接到服务器时，要在IRC客户端中显示的IRC 标题可以在/etc/ircd/ircd.motd文件中设置 。

对配置文件进行必要的更改后，可以使用以下命令重新启动IRC服务器： 

sudo systemctl restart ircd-irc2.service

参考

您可能还有兴趣了解Ubuntu Repository中提供的其他IRC服务器。它包括 ircd-ircu和 ircd-hybrid。
