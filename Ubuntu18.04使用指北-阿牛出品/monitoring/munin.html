安装

在server01上安装Munin之前，需要安装apache2 。默认配置适用于运行munin服务器。有关更多信息，请参阅HTTPD - Apache2 Web服务器。

首先，在server01上安装munin。在终端输入： 

sudo apt install munin

现在在server02上安装munin-node软件包： 

sudo apt安装munin-node

组态

在SERVER01编辑/etc/munin/munin.conf增加对IP地址Server02上： 

##首先是我们的“正常”主持人。
[Server02上]
       地址172.18.100.101

将server02和172.18.100.101替换为服务器的实际主机名和IP地址。

接下来，在server02上配置munin-node。编辑 /etc/munin/munin-node.conf以允许server01访问： 

允许^ 172 \ .18 \ .100 \ .100 $

用您的munin服务器的IP地址替换^ 172 \ .18 \ .100 \ .100 $。

现在重新启动server02上的munin-node以使更改生效： 

sudo systemctl restart munin-node.service

最后，在浏览器中转到http：// server01 / munin，您应该看到指向显示来自磁盘，网络，进程和系统的标准munin-plugins信息的漂亮图表的链接。

由于这是一个新安装，图表可能需要一些时间才能显示任何有用的内容。

附加插件

该穆宁-插件-额外的包中包含性能检查额外服务，如DNS，DHCP，桑巴等要安装该软件包，从一个终端输入： 

sudo apt install munin-plugins-extra

确保在服务器和节点计算机上安装软件包。
