存档轮换

Shell Scripts 中的shell脚本仅允许七个不同的归档。对于数据不经常更改的服务器，这可能就足够了。如果服务器具有大量数据，则应使用更复杂的轮换方案。

  • 旋转NFS档案
  • 磁带机

旋转NFS档案

在本节中，将稍微修改shell脚本以实现祖父 - 父子旋转方案（每月每周 - 每日）：

  • 轮换将在周日至周五进行每日备份。

  • 星期六每周进行一次备份，每月进行四次备份。

  • 将每月备份在第一个月基础上，如果当月是奇数还是偶数回转二每月备份的完成。

这是新脚本： 

＃！/斌/庆典
####################################
＃
＃使用备份到NFS挂载脚本
#grangfather-father-son轮换。
＃
####################################

＃备份什么 
backup_files =“/ home / var / spool / mail / etc / root / boot / opt”

＃备份到哪里。
DEST = “/ MNT /备份”

＃存档文件名的设置变量。
day = $（日期+％A）
hostname = $（hostname -s）

＃查找1-4个月的哪一周。
day_num = $（日期+％ -  d）
if（（$ day_num <= 7））; 然后
        week_file = “$主机名week1.tgz”
elif（（$ day_num> 7 && $ day_num <= 14））; 然后
        week_file = “$主机名week2.tgz”
elif（（$ day_num> 14 && $ day_num <= 21））; 然后
        week_file = “$主机名week3.tgz”
elif（（$ day_num> 21 && $ day_num <32））; 然后
        week_file = “$主机名week4.tgz”
科幻

＃查找月份是奇数还是偶数。
month_num = $（日期+％m）
month = $（expr $ month_num％2）
如果[$ month -eq 0]; 然后
        month_file = “$主机名month2.tgz”
其他
        month_file = “$主机名month1.tgz”
科幻

＃创建存档文件名。
如果[$ day_num == 1]; 然后
	ARCHIVE_FILE = $ month_file
elif [$ day！=“Saturday”]; 然后
        ARCHIVE_FILE = “$ hostname- $ day.tgz”
其他 
	ARCHIVE_FILE = $ week_file
科幻

＃打印开始状态消息。
echo“将$ backup_files备份到$ dest / $ archive_file”
日期
回声

＃使用tar备份文件。
tar czf $ dest / $ archive_file $ backup_files

＃打印结束状态消息。
回声
echo“备份完成”
日期

＃$ dest中文件的长列表，用于检查文件大小。
ls -lh $ dest /

可以使用与执行脚本中相同的方法执行脚本。

在发生灾难时，最好将备份媒体带到异地。在shell脚本示例中，备份媒体是另一个提供NFS共享的服务器。很可能将NFS服务器带到另一个位置是不切实际的。根据连接速度，可以选择通过WAN链接将存档文件复制到另一个位置的服务器。

另一个选择是将存档文件复制到外部硬盘驱动器，然后可以将其从外部移动。由于外部硬盘的价格持续下降，因此为每个存档级别使用两个驱动器可能具有成本效益。这将允许您将一个外部驱动器连接到备份服务器，将一个外部驱动器连接到另一个位置。

磁带机

可以使用连接到服务器的磁带驱动器而不是NFS共享。使用磁带驱动器可以简化存档轮换，并且还可以使介质更容易离开现场。

使用磁带驱动器时，不需要脚本的文件名部分，因为数据直接发送到磁带设备。需要一些操作磁带的命令。这是使用mt（cpio包的磁带控制实用程序部分）完成的。

以下是修改为使用磁带机的shell脚本： 

＃！/斌/庆典
####################################
＃
＃备份到磁带机脚本。
＃
####################################

＃备份什么 
backup_files =“/ home / var / spool / mail / etc / root / boot / opt”

＃备份到哪里。
DEST = “的/ dev / ST0”

＃打印开始状态消息。
echo“将$ backup_files备份到$ dest”
日期
回声

＃确保磁带倒带。
mt -f $ dest倒带

＃使用tar备份文件。
tar czf $ dest $ backup_files

＃倒带并弹出录像带。
mt -f $ dest rewoffl

＃打印结束状态消息。
回声
echo“备份完成”
日期

SCSI磁带机的默认设备名称是/ dev / st0。使用适用于您系统的设备路径。

从磁带驱动器恢复与从文件恢复基本相同。只需回放磁带并使用设备路径而不是文件路径。例如，要将/ etc / hosts文件还原到/ tmp / etc / hosts： 

mt -f / dev / st0倒带
tar -xzf / dev / st0 -C / tmp等/ hosts
