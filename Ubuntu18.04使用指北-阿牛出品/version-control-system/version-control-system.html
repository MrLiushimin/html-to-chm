版本控制系统

版本控制是管理信息变更的艺术。对于程序员来说，它一直是一个重要的工具，程序员通常会花时间对软件进行小的更改，然后在第二天撤消这些更改。但版本控制软件的实用性远远超出了软件开发领域的范围。在任何地方，您都可以找到使用计算机管理经常更改的信息的人，还有版本控制的空间。
