市场

Bazaar是由Ubuntu背后的商业公司Canonical赞助的新版本控制系统。与仅支持中央存储库模型的Subversion和CVS不同，Bazaar还支持 分布式版本控制，使人们能够更有效地进行协作。特别是，Bazaar旨在最大限度地提高社区参与开源项目的水平。

安装

在终端提示符下，输入以下命令以安装 bzr： 

sudo apt install bzr

组态

要向bzr介绍自己，请使用whoami命令，如下所示： 

$ bzr whoami'Joe Doe <joe.doe@gmail.com>'

学习义卖市场

Bazaar 默认将附带的文档安装到 / usr / share / doc / bzr / html中。本教程是一个很好的起点。该 BZR命令还带有内置的帮助： 

$ bzr帮助

要了解有关foo命令的更多信息： 

$ bzr帮助foo

Launchpad集成

虽然Bazaar作为独立系统非常有用，但它与Launchpad，Canonical使用的协作开发系统以及更广泛的开源社区管理和扩展Ubuntu本身具有良好的可选集成。有关Bazaar如何与Launchpad一起使用以在开源项目上进行协作的信息，请参阅 http://bazaar-vcs.org/LaunchpadIntegration。
