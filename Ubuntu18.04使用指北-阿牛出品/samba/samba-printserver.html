打印服务器

Samba的另一个常见用途是将其配置为在Ubuntu服务器上共享本地或通过网络安装的打印机。与文件服务器类似，此部分将配置Samba以允许本地网络上的任何客户端使用已安装的打印机，而不会提示输入用户名和密码。

有关更安全的配置，请参阅保护文件和打印服务器。

安装

在安装和配置Samba之前，最好已经安装了CUPS 。有关详细信息， 请参阅CUPS - 打印服务器

要从终端安装samba软件包，请输入： 

sudo apt安装samba

组态

安装samba后编辑/etc/samba/smb.conf。将工作组 属性更改为适合您的网络的属性，并将安全性更改为 用户： 

   workgroup =示例
   ...
   security = user

在[printers]部分中，将guest ok选项更改为yes： 

   可浏览=是的
   客人ok =是的

编辑smb.conf后重新启动Samba： 

sudo systemctl restart smbd.service nmbd.service

默认的Samba配置将自动共享安装的所有打印机。只需在Windows客户端上本地安装打印机即可。
