pam_motd

登录到Ubuntu服务器时，您可能已经注意到信息性的当日消息（MOTD）。使用几个包获取并显示此信息：

  • landscape-common：提供landscape-client的核心库，管理具有Landscape（专有）的系统。然而，该软件包还包括landscape-sysinfo实用程序，它负责显示涉及cpu，内存，磁盘空间等的核心系统数据。例如： 

    
	  系统负载：0.0进程：76
	  /的使用率：3.11GB用户的30.2％登录：1
	  内存使用情况：eth0的IP地址为20％：10.153.107.115
	  掉期使用：0％

	  绘制此数据并在https://landscape.canonical.com/管理此系统

    您可以随时手动运行landscape-sysinfo。

  • update-notifier-common：提供有关可用包更新，即将发生的文件系统检查（fsck）和所需重新启动的信息（例如：在内核升级之后）。

pam_motd根据脚本前面的数字按顺序执行/etc/update-motd.d中的脚本。脚本的输出写入/ var / run / motd，保持数字顺序，然后与/etc/motd.tail连接。

您可以将自己的动态信息添加到MOTD。例如，要添加本地天气信息：

  • 首先，安装weather-util包： 

    sudo apt install weather-util

  • 该天气工具从国家海洋和大气管理局和预测从国家气象局使用METAR数据。为了找到本地信息，您需要4个字符的ICAO位置指示器。这可以通过浏览国家气象局网站来确定 。

    虽然国家气象局是美国政府机构，但全世界都有气象站。但是，可能无法获得美国境外所有地点的当地天气信息。

  • 创建/ usr / local / bin / local-weather，一个简单的shell脚本，将 天气与您当地的ICAO指标结合使用：

  • 使脚本可执行： 

    sudo chmod 755 / usr / local / bin / local-weather

  • 接下来，创建一个符号链接到/etc/update-motd.d/98-local-weather： 

    sudo ln -s / usr / local / bin / local-weather /etc/update-motd.d/98-local-weather

  • 最后，退出服务器并重新登录以查看新的MOTD。

您现在应该收到一些有用的信息，以及一些可能不太有用的当地天气信息。希望本地天气示例展示了pam_motd的灵活性 。
